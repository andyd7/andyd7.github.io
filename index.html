// units covered so far

A few things I've learned in AP CSA

Some of the core ideas from this year's units that I've used in my projects and FRQ practice.

Unit 2 — Using Objects

Classes & Objects

Learned how to design classes with constructors, instance variables, and methods to model real things in code, and how to call methods on objects to make them do work.

Unit 4 — Iteration

Loops & Iteration

Practiced writing while and for loops, including nested loops, to repeat tasks and process data without writing the same code over and over.

Unit 6 — Array

Arrays & ArrayLists

Worked with arrays and ArrayList to store collections of data, then traversed, searched, and updated them with loops.

Unit 9 — Inheritance

Inheritance & Polymorphism

Explored how subclasses extend a parent class, override methods, and how polymorphism lets different objects respond to the same method call in their own way.
